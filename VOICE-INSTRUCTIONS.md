# Голосовые сообщения из Telegram

Когда из Telegram приходит сообщение с `attachment_kind="voice"` или
`attachment_kind="audio"` (в теге `<channel source="telegram" ...>`):

1. Вызови `download_attachment` с `attachment_file_id` — получишь локальный путь.
2. Прогони файл через транскрайбер и получи текст:
   ```
   ~/.claude/buildserver/transcribe.sh "<путь-из-шага-1>"
   ```
3. Дальше работай с распознанным текстом как с обычным запросом пользователя.
4. Если транскрипция вернула ошибку (нет ключа/сети) — коротко сообщи об этом
   через `reply`, не угадывай содержимое.

Добавь содержимое этого файла в `CLAUDE.md` папки, где работает билд-сервер
(`BUILD_DIR`), чтобы Claude знал про этот шаг.
